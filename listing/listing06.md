Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Выведет:
  [3 2 3]
В функцию modifySlice мы передаем копию дескриптора слайса, который содержит три
поля: указатель на элемент нижележащего масива, длина слайса, вместимость слайса.
Таким образом слайс описывает сегмент массива. При изменении нулевого элемента
слайса мы изменям элемент общего нижележащего массива. Так как длина и
вместимость слайса равны, при добавлении элемента в слайс в локальную переменную
функции i запишется новый дескриптор слайса с указателем на новый увеличенный
нижележащий массив (в него скопируются элементы из старого и добавится новый
элемент). Далее мы изменяем уже новый слайс (точнее его новый нижележащий массив)
и добавляем новый элемент (вместимость равная 6 позволяет добавить элемент без
увеличения нижележащего масива).
При увеличении массива он удваивается пока не перейдет порог в 256 элементов,
затем размер увеличения плавно снижается к 1.25.

https://github.com/golang/go/blob/release-branch.go1.18/src/runtime/slice.go#L188
```

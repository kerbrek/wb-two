Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Выведет:
  2
  1
Отложенная функция может изменять возвращаемые именованные параметры, которые
она замыкает.
defer откладывает вызов функции до момента возврата из окружающей функции (либо
из-за того, что окружающая функция выполнила оператор return, достигла конца
тела своей функции, либо из-за паники горутины). Аргументы отложенной функции
вычисляются немедленно, а сами отложенные функции вызываются непосредственно
перед возвратом из окружающей функции в порядке Last In First Out. Если
окружающая функция возвращает результат в операторе return, отложенные функции
выполняются после того, как результирующие параметры установлены оператором
return, но до того, как функция вернется к вызывающей стороне. Если окружающая
функция возвращает результат через именованные параметры, которые отложенная
функция замыкает, то отложенная функция может изменить результирующие параметры
до того, как они будут возвращены.
https://go.dev/ref/spec#Defer_statements
```
